# Quelque truc gaming à installer 

Sur Archlinux installer certaine librairies qui peuvent être manquantes :

``sudo pacman -S mesa lib32-mesa vulkan-radeon mesa-vdpau lib32-vulkan-radeon lib32-mesa-vdpau libva-mesa-driver lib32-libva-mesa-driver vulkan-icd-loader lib32-vulkan-icd-loader``

activer goverlay pour steam avec la commande suivante :  

``gamemoderun mangohud %command%``

Pour project ascension supprimer le fichier suivant pour le lancer :

``DivxTac.dll``
